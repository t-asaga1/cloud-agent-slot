# reference — 演出用画像の参考画像置き場

演出用の画像を生成する際に AI・デザイン作業の参考にする画像(キャラクターデザイン・背景の雰囲気など)をアップロードするフォルダです。
入稿の一般的な手順は `incoming/README.md` と `docs/ASSET_GUIDELINES.md` を参照してください。

**このフォルダの画像はアプリに直接取り込まれません**(参考資料扱い)。生成した演出用画像を取り込む際は、従来どおり `incoming/` 直下の用途別フォルダ(`video/` など)を使います。

## フォルダ構成

### キャラクターの参照画像(`キャラクター/`)

| フォルダ | 用途 |
| --- | --- |
| `義経/` | 主人公・義経の参照画像 |
| `静/` | 静(静御前)の参照画像 |
| `弁慶/` | 弁慶の参照画像 |
| `頼朝/` | 頼朝の参照画像 |
| `黒幕/` | 黒幕の参照画像 |
| `その他/` | 上記以外のキャラクター・小物などの参照画像 |

### キャラクター設定資料(`設定資料/`)

Seedance 2.0 へ渡すために GPT Image 2 で生成した設定資料画像(キャラごとに「顔ヘッドショット」+「全身」の 2 枚。無地グレー背景・均一照明・真正面)。
動画生成時はこのペアを参照画像として渡す(渡し方のルールは `docs/SEEDANCE_GUIDELINES.md` を参照)。
再生成は `scripts/gen_character_sheets.mjs`(入力は `キャラクター/` のユーザー入稿参考画像)。

### 背景の参照画像(`背景/`)

| フォルダ | 対応するステージ(`STAGE_IDS`) |
| --- | --- |
| `義経背景/` | 通常時・義経背景 |
| `弁慶背景/` | 通常時・弁慶背景 |
| `静背景/` | 通常時・静背景 |
| `夕方背景/` | 通常時・夕方背景 |
| `前兆背景/` | 前兆中背景 |
| `下位AT/` | AT(下位)の小役/バトルステージ |
| `上位AT/` | 上位 AT の小役/バトルステージ |
| `その他/` | 上記に当てはまらない背景(エンディング・タイトルなど) |

## 使い方

1. 該当フォルダに画像を置く(ファイル名は内容がわかるものに。日本語可)。
2. 必要なら「どのシーン・どの演出向けか」を AGENT への指示文に添える。
3. コミット & プッシュ(GitHub Web UI の `Add file → Upload files` で OK。25 MiB 超は GitHub Desktop / git コマンドで)。

※ 各フォルダ内の `.gitkeep` は空フォルダを Git 管理するためのファイルです(画像を入れた後もそのままで OK)。
