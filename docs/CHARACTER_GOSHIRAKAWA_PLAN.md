## 1. 決定事項(2026-07-17 ユーザー指示)

- 黒幕であるラスボスは **後白河法皇** とする。
- キャラデザインは他キャラと同様に **GPT Image 2(fal.ai `openai/gpt-image-2/edit`)** で作成する。
- デザイン条件:
  1. ユーザー添付の実物写真(座像)の**衣装感(白い法衣 + 袈裟)と剃髪(坊主)は維持**する。
  2. 年齢は実物より**もっと若く、力強く、禍々しい**感じにする。
  3. 属性は**赤い氷**(義経 = 青い炎 / 頼朝 = 紫の雷 に対する対比。上位 AT が雪原であることと整合)。

## 2. 制作フロー(他キャラとの違い)

既存 4 キャラは「ユーザー入稿の立ち絵 → 設定資料化」だったが、黒幕は**デザイン自体をゼロから生成**する。

## 4. プロンプト案(全身デザイン画)

### 共通ベース(全案に含める)

```
パチスロ「義経物語」のラスボス・後白河法皇のキャラクターデザイン(全身立ち絵)を作成してください。

【画風】@Image2 と同じフォトリアルな 3D ゲームキャラクターレンダー調。質感・ライティング・頭身を
@Image2 に合わせる(デザイン自体は @Image2 から流用しない)。
【衣装の基本】@Image1 の実物写真を参照: ゆったりした白い法衣(僧衣)+ 肩から斜めに掛けた
赤茶(柿渋色)の袈裟 + 剃髪(坊主頭)。この「白い法衣 + 斜め掛けの袈裟 + 剃髪」の構成は必ず維持する。
【顔・年齢】@Image1 の顔立ち・年齢は参照しない。若く力強い僧形の男。血の気のない蒼白な肌、
赤く光る鋭い瞳、冷酷で禍々しい微笑。
【属性】「赤い氷」。半透明の深紅の氷の結晶が肩・袖口・裾に張り付き、法衣の裾は凍りついて
足元から赤い冷気(フロストミスト)が漂う。氷は血のように深い赤で、内側から不気味に発光する。
炎や雷のエフェクトは出さない。
【構図】自然な直立の立ちポーズ、真正面、頭からつま先まで全身が完全に収まる。
背景は無地の明るいグレー、影のない均一なスタジオライティング。文字・ロゴ・枠・透かしは入れない。
```

### 案 A(推奨): 「若き覇王」— 30 歳前後・力強さ重視

ベースに以下を追加:

```
【追加指定】年齢は 30 歳前後。長身で肩幅のある堂々とした体格。眉は鋭く、目つきは獲物を見下ろす
ように冷たい。額に赤い氷の結晶でできた第三の目のような紋様が浮かぶ。左手に深紅の氷でできた
大数珠を提げ、右手は軽く開いて掌に小さな赤い氷晶を浮かべている。袈裟の縁と首元に金の刺繍で
菊の文様をあしらい、法皇(治天の君)の格を示す。
```

- ねらい: 「力強く、禍々しく」を最も素直に出す案。頼朝(白銀の長髪・毛皮)と並んだときの対比が明確(剃髪 + 白装束 + 赤い氷)。
- 額の紋様は頼朝の額の紋様と「王家の血筋」で対になる意匠(不要なら削る)。

### 案 B: 「壮年の巨悪」— 40 代・威厳と底知れなさ重視

ベースに以下を追加:

```
【追加指定】年齢は 40 代前半。「若返らせた後白河」とわかる面影を残しつつ、頬はこけ、
彫りの深い顔立ちに深い威厳を宿す。表情は薄笑いで目だけが笑っていない。右手に赤い氷が
螺旋状に巻き付いた黒漆の錫杖(頭部の遊環も深紅の氷でできている)を持ち、左手に数珠を提げる。
白い法衣の裾は長く床へ広がり、裾の先が凍って赤い氷の棘になっている。
```

- ねらい: 「若く」の度合いを控えめにし、実在の後白河のイメージ(老獪な黒幕)を残す案。武器は錫杖でラスボスらしい迫力を出す。

### 案 C: 「傀儡師」— 20 代後半・妖しさ重視(琵琶持ち)

ベースに以下を追加:

```
【追加指定】年齢は 20 代後半。中性的で妖しく整った顔立ち、切れ長の目。表情は恍惚とした
薄笑いで狂気を感じさせる。背に深紅の氷でできた琵琶を負い、右手にその撥(ばち)を持つ
(実物写真で膝に置かれた琵琶のモチーフを武器化したもの)。指先から赤い氷の糸が数本、
操り人形の糸のように垂れて足元の冷気へ消えていく。
```

- ねらい: 史実の後白河(今様狂い・「日本一の大天狗」)のモチーフを活かした案。「源平を裏で操っていた黒幕」の演出(糸)と、雪原ステージで琵琶をかき鳴らす戦闘演出への発展性がある。
