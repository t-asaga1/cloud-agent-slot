## 1. 確定した仕様(固有予告 3 = 会話予告・3 段階)

旧「コメント予告」案(`docs/YOKOKU_PRODUCTION_PLAN.md`「10.」)を置き換える。

- **レバオンで一言目**: 会話ウィンドウが**画面左**に表示(背景は見える)。
- **第 1 停止で二言目**が発生することがある: ウィンドウは**画面右**(背景は見える)。
- **第 3 停止で全画面**になることがある: 背景は見えない。キャラ中央の決めポーズ + 大台詞。
- 一言目・二言目は**両方とも弱扱い**。小役が成立している場合は二言目まで行く。
- **第 3 停止の全画面 = 強予告**。
- セリフは仮でよい(全画面は「チャンスだ(仮)」)。**画像にはテキストを焼き込まず、
  台詞はアプリ側テキスト描画**(台詞テーブルは組込み時に `direction.ts` へ)。

## 2. 生成した 13 枚(`incoming/yokoku/` へ原本コミット済み)

いずれも 1536x1024(3:2)。内蔵 GenerateImage(FAL 課金なし)。

| ファイル | 内容 |
|---|---|
| `会話予告_枠.png` | 会話ウィンドウ枠単体(和風・漆黒 + 金縁 + 桜と雲の飾り + 左上ネームプレート。中は無地) |
| `義経_会話予告_一言目.png` ほか 4 キャラ | 枠の左 1/3 にバストアップ。台詞スペース(右 2/3)側を向いて話すポーズ |
| `義経_会話予告_二言目.png` ほか 4 キャラ | 左右反転構成(キャラが右 1/3・左を向く)。表情は一言目より強め |
| `義経_会話予告_全画面.png` ほか 4 キャラ | 枠なし・キャラ中央の決めポーズ(義経 = 抜刀 / 頼朝 = 軍配 / 静 = 二枚扇の舞 / 弁慶 = 薙刀の薙ぎ)+ キャラ別カラーのエネルギー背景 + 大台詞用の空きスペース(左上・右下) |

- 生成手順: 設定資料ペア(顔 + 全身)+ 枠単体を参照 → 一言目 → 二言目は一言目を参照連鎖

- **頼朝の全画面は 1 回目で顔・髪がドリフト(白髪の長髪化)** →「黒髪短髪・烏帽子・
  顔を変えない」を明示して作り直した(採用 = v2 のみコミット)。
- ユーザー確認用に、仮セリフを PIL(Noto Serif/Sans CJK)で焼き込み + ステージ背景と
