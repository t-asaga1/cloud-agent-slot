## 3. 生成のコツ(以後の予告にも適用。YOKOKU_PRODUCTION_PLAN 12.3 に記録済み)

1. 参照画像に設定資料ペア(`incoming/reference/設定資料/静_顔.png` / `静_全身.png`)と
   背景参考(`incoming/reference/背景/静背景/清水寺_側面.png`)を渡す。
2. **2 枚目以降は 1 枚目の生成画像を参照画像に連鎖させる**(プロンプトに
   「参照と同一、変更点は◯◯のみ」と明記)。今回、顔・衣装(立烏帽子・瑠璃色の首飾り)・
   背景(本堂・欄干・夕焼けの山並み)・ライティングの一貫性が 4 枚とも保てた。
3. 図柄合成用の画像(空)は「中央を空けて雲や被写体を置かない(a clean open area so
   that a game symbol can be composited)」を明記。
4. 背景一体化(書き割り禁止・環境光一致・足元の影)は AGENT #078 の教訓を
   静止画プロンプトにもそのまま適用。
