## 計測方法

- 実行コマンド: `npm run sim -- <ゲーム数> <シード>`(`scripts/run_simulation.ts` → `src/core/simulate.ts`)
- 1G の実行: `src/core/play.ts` の `playGame` = 役抽せん(`drawRole`)→ 打ち方ポリシー → リール制御(`resolveSpin`)→ ステートマシン(`advanceGame`)
- 打ち方(確定 26):
  - 通常時 = **左第一・適当押し**(押下位置は全リール一様ランダム。チェリー・スイカ・リーチ目は取りこぼしあり)
  - AT 中 = **ナビ遵守**(押し順ベルは中第一 = 斜め揃い 13 枚。ベル以外は左第一・適当押しのままなのでレア役の取りこぼしは通常時と同様に発生)
- 区間の分類(ゲーム開始時点のフェーズ):
  - 通常区間 = 通常 / 前兆 / 連続演出(連続演出成功の告知ゲームまで)
  - AT 区間 = AT(小役・バトル)/ エンディング
- 指標の定義は `src/core/simulate.ts` のヘッダーコメント参照(機械割 = 総払出 ÷ 総投入、コイン持ち = 通常区間 50 枚あたりゲーム数、初当り = 通常区間 G 数 ÷ AT_START 回数、連チャン数 = セット数)
