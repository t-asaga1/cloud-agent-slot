# STEP 1 完了時点の動作確認手順(ユーザー向け)

作成: AGENT #024(2026-07-11)
対象: STEP 1(リール制御の本対応)完了時点の main ブランチ。
所要時間の目安: 自動チェック 1〜2 分 + ブラウザ確認 5〜10 分。

## 0. 前提(環境)

- Node.js 22 系 + npm(`node -v` / `npm -v` で確認)
- リポジトリの main ブランチを最新化(GitHub Desktop で Fetch / Pull)

## 1. 自動チェック(テスト・lint・ビルド)

リポジトリのフォルダでターミナルを開き、以下を順に実行する。

```
npm install        # 初回、または pull 後に package.json が変わっていたら
npm test           # 期待: Tests 125 passed (125)、skip 0
npm run lint       # 期待: エラーなし
npm run build      # 期待: エラーなしで dist/ が生成される
```

`npm test` には全役 × 全 20³ 押下位置 × 押し順 6 通りの停止制御網羅テストが含まれるため、
これがグリーンなら停止制御ロジックの仕様適合(引き込み・蹴飛ばし・取りこぼし)は機械的に保証される。
ブラウザ確認は「見た目と操作の確認」が目的。

## 2. アプリの起動

```
npm run dev
```

表示された URL(デフォルト http://localhost:5173)をブラウザで開く。
画面: 筐体 + 背景動画 + リール窓(左)、操作パネル + ステータス + ゲーム履歴(右)。

### 操作パネルの見方

| 項目 | 内容 |
|---|---|
| ステージ | 背景動画の手動切替(9 種。状態遷移との連動は STEP 2) |
| 押し順 | 第一〜第三停止の順(6 通り) |
| 目押し | 適当押し(ランダム)/ 赤7 狙い(全リール)/ DDT(左黒バー狙い) |
| 成立役 | 「抽せん(通常)」= 実確率で抽せん / 各役の強制指定 = 停止形確認のデモ用 |
| レバーオン(1G 消化) | 1 ゲーム進行。出目・揃ったライン(金枠)・払出・履歴が更新される |

## 3. ブラウザ確認チェックリスト

各項目とも「成立役」を強制指定 → レバーオンを数回押して出目を確認する。
揃ったライン上の図柄は**金枠でハイライト**され、ステータス欄に「揃ったライン」が表示される。

### 3-1. 基本役(押し順で挙動が変わる)

| # | 設定 | 期待する結果 |
|---|---|---|
| 1 | 成立役=押し順ベル(強制)、押し順=左→中→右 | 毎回**上段**にベル揃い・払出 1 枚(押し順不正解) |
| 2 | 成立役=押し順ベル(強制)、押し順=中→右→左 など中・右第一 | 毎回**斜め**(右下がり/右上がり)にベル揃い・払出 13 枚(押し順正解) |
| 3 | 成立役=リプレイ(強制)、押し順 6 通り × 目押し=適当押し | **どの押し順でも毎回**横 or 斜めのいずれかに揃う(100% 引き込み。取りこぼしなし)。次ゲームは BET 不要(収支が減らない) |
| 4 | 成立役=ハズレ(強制)、適当押しで 10G ほど | 何も揃わず、左リール窓内にチェリーも出ない(クリーンなハズレ目) |

### 3-2. レア役(取りこぼしあり)

| # | 設定 | 期待する結果 |
|---|---|---|
| 5 | 成立役=角チェリー(強制)、適当押しで 10G ほど | 揃うときは左リールの**上段 or 下段**にチェリー(2 枚)。引き込めない押下位置では出目=ハズレ(取りこぼし。ステータスに「取りこぼし/未成立」) |
| 6 | 成立役=中段チェリー(強制)、適当押しで 10G ほど | 揃うときは左リール**中段**にチェリー(2 枚)。取りこぼしあり |
| 7 | 成立役=弱スイカ(強制)、適当押しで 10G ほど | 揃うときは**斜め優先**でスイカ揃い(3 枚)。取りこぼしあり |
| 8 | 成立役=強スイカ(強制)、適当押しで 10G ほど | 揃うときは**平行(横)優先**でスイカ揃い(3 枚)。取りこぼしあり |
| 9 | 成立役=チャンス目(強制)、適当押しで 10G ほど | スイカが**テンパイはずれ**の形(スイカ 2 個が載るがスイカは揃わない)で 3 枚。取りこぼしあり |

### 3-3. リーチ目と DDT(目押しモード)

| # | 設定 | 期待する結果 |
|---|---|---|
| 10 | 成立役=リーチ目(強制)、**目押し=赤7 狙い** | **毎回**いずれかの有効ラインに赤7 が 3 つ揃う(3 枚) |
| 11 | 成立役=リーチ目(強制)、目押し=適当押し | ほとんど取りこぼし(出目=ハズレ・払出なし)。たまに揃うのは正常(赤7 を引き込める位置で押せた場合) |
| 12 | 成立役=ハズレ(強制)、**目押し=DDT**、押し順=左→中→右 | 左リールの黒バーが**下段付近**に停止する |
| 13 | 成立役=弱 or 強スイカ(強制)、目押し=DDT、押し順=左→中→右 | 左リールにスイカが必ず窓内へスベってくる(DDT でスイカを察知できる) |

### 3-4. 通常抽せんでの流し確認

| # | 設定 | 期待する結果 |
|---|---|---|
| 14 | 成立役=抽せん(通常)、適当押し、押し順=左→中→右で 30G ほど | ベル(1 枚)が約 2/3、リプレイ・ハズレがそれぞれ約 1/7 の頻度感。レア役はまれ。収支は緩やかに減る(通常時 ≒ −1.8 枚/G)。ブラウザの開発者ツール(F12)の Console にエラーが出ない |
