**1 サブステップ = 1 AGENT = 1 PR** の粒度へ分割した(AGENT #015 で計画)。

各サブステップは「テスト・lint・ビルドがグリーンの状態」で終わるよう設計している
(途中サブステップでは停止制御が暫定のままでもよいが、壊れた状態でのマージは不可)。

## 進め方のルール(再掲)

- 1 AGENT に対するやり取りは 1 回。大きい STEP(2〜3)は「実装 → 指摘反映」で 2 AGENT に分けてもよい
- STEP 1 は 1a〜1f のサブステップへ分割済み。**1 サブステップ = 1 AGENT = 1 PR** で順番に進める
- 各 STEP 完了ごとに PR を 1 本ずつマージする(`docs/HANDOVER.md` が競合しやすいため)
- 数値(確率・配当・配列)は必ず SPEC.md と実装・テストを一致させる
